# Дыхание — PWA приложение

Личное приложение для дыхательных практик.

## Техники
- **Вим Хоф** — 3 раунда по 30 вдохов, задержки 60/90/120 сек
- **Хьюберман** — physiological sigh, 5 минут
- **Квадрат** — box breathing 4-4-4-4, 5 минут
- **Молчанов** — пирамида 5→15→5 секунд

## Функции
- Таймеры с аудио-сигналами (Web Audio API)
- Вибрация (телефон)
- Визуальная анимация дыхания (орб)
- Трекер серии дней
- Утреннее напоминание (браузерные уведомления)
- PWA — работает офлайн, можно добавить на главный экран

---

## Деплой на GitHub Pages (бесплатно, без карты)

1. Создай аккаунт на github.com
2. Создай новый репозиторий (например `breath`)
3. Загрузи все файлы (`index.html`, `manifest.json`, `icon.svg`, `sw.js`)
4. Зайди в Settings → Pages → Source: Deploy from branch → main → / (root)
5. Через минуту приложение будет доступно по адресу:
   `https://ВАШ_НИК.github.io/breath`

## Деплой на Vercel (бесплатно, без карты)

1. Создай аккаунт на vercel.com через GitHub
2. New Project → Import репозиторий
3. Deploy — готово

## Добавить на главный экран iPhone

1. Открой приложение в Safari
2. Нажми кнопку "Поделиться" (квадрат со стрелкой)
3. "Добавить на экран «Домой»"
4. Теперь запускается как нативное приложение

## Вибрация на Apple Watch

Вибрация работает автоматически когда телефон в руке.
Для вибрации на часах без телефона — нужно нативное приложение WatchOS (это отдельная разработка).
